# Aprendizado de Markdown

Repositório para aprendizado da linguagem Markdown durante o curso de Git e GitHub ministrado pelo Prof. Gustavo Guanabara.

# Formatações:

## Itálico:

Para criar conteúdo em itálico, utilizamos as marcas \* ou \_. Exemplo:

Eu sou o *Vitor* e este é meu perfil no _GitHub_!

## Negrito:

Para criar conteúdo em negrito, utilizamos as marcas \** ou \__. Exemplo:

Eu sou o **Vitor** e este é meu perfil no __GitHub__!

## Riscado (strike):

Para riscar um texto, utilizamos a marca \~~. Exemplo:

Este ~~texto~~ está riscado.

## Formatações misturadas:

Para criar formatações misturadas, utilizamos as marcas de forma hierárquica. Exemplo:

Eu sou o _**Vitor**_ e este é meu perfil no _**GitHub**_.

# Listas:

## Listas numeradas:

Para criar uma lista numerada, colocamos o número seguido de um ponto. Os itens vão seguir a sequência. Exemplo:

1. Brasil
2. Argentina
3. Venezuela
4. Chile

## Listas com marcadores:

Para criar uma lista com marcadores, utilizamos a marcação de \* ou \- seguida de um espaço. Exemplo:

* Ir ao mercado
    * Comprar carne
    * Comprar batata
* Ir a farmácia

## Listas de tarefas:

Para criar uma lista de tarefas, utiliza-se a marcação de \- \[] ou \- \[x] para itens marcados. Exemplo:

- [ ] pão
- [x] leite
- [ ] manteiga

# Títulos:

Para criar um título, podemos usar de uma \# até seis \###### para cada nível de título. Exemplo:

# Título 1
## Título 2
### Título 3

# Linhas horizontais:

Para criar linhas horizontais, utilizamos três traços \--- ou três asteriscos \***. Exemplo:

# Título principal
***
Teste

# Imagens:

Para inserir uma imagem, utilizamos \!\[]\(), onde a descrição da imagem fica entre os colchetes e a fonte da imagem fica entre os parênteses. Exemplo:

![Calculadora](img/calculadora.svg)

# Links:

Para inserir um link, utilizamos a mesma marcação das imagens, porém removemos o ponto de exclamação. Exemplo:

[Google](https://www.google.com)

# Trechos de código:

## Trechos de código de uma linha:

Para inserir códigos de uma linha, utilizamos \`. Exemplo:

`alert('Olá, mundo!')`

## Trechos de código com mais de uma linha:

Para códigos com mais linhas, utilizamos \```. Exemplo:

```
alert('Olá, mundo!');
console.log('Estou aprendendo Markdown!');
```

# Citações:

## Citações externas:

Para citações externas, podemos usar \>. Exemplo:

Como **Cora Coralina** disse:

> Não lamente o que podia ter e se perdeu por
> caminhos errados e nunca mais voltou.

## Citações de usuários:

Para mencionar usuários, utilizamos \@. Exemplo:

Meu perfil é @vitorgabsouza

Para mencionar Issues, utilizamos \# seguida pelo número da Issue.

# Emojis: 

Para utilizar emojis, utilizamos \::. Exemplo:

Boas iniciativas merecem aplausos :clap:

# Tabelas:

Para utilizar tabelas, utilizamos \| e \- de forma organizada. Exemplo:

Produto | Categoria | Preço
:--- | :---: | ---:
Coxinha de Frango | Aperitivo | R$ 5,00
Macarrão ao Molho de Tomate | Prato Principal | R$ 15,00
Sorvete de Creme | Sobremesa | R$ 4,50

## Alinhamentos

\--- Formatação padrão (esquerda)

\:--- Esquerda

\:---: Centralizado

\---: Direita
